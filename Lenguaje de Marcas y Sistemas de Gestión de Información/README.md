# Proyecto de Lenguajes de Marcas y Sistemas de Gestión de Información (LMSGI)

Este repositorio contiene las prácticas y fases del proyecto desarrollados durante el módulo de Lenguajes de Marcas. El trabajo abarca desde la estructuración semántica de la información y planificación de proyectos hasta el diseño visual avanzado de interfaces web.

## Herramientas Utilizadas

* HTML5: Lenguaje de marcado para la estructuración semántica y accesible del contenido web.
* CSS: Hojas de estilo en cascada para la maquetación, diseño visual y adaptabilidad (Responsive Web Design).
* Editores de Código: Uso de entornos de desarrollo (IDE) y editores de texto plano para la escritura de código limpio y estructurado.

---

## Bloques Temáticos y Prácticas

### 1. Fases del Proyecto Web
Documentación y desarrollo estructural de un sitio web desde su concepción hasta su maquetación básica:
* Fase de Planificación: Definición de la arquitectura de la información, esquema de navegación y requerimientos del proyecto.
* Estructuración Semántica: Creación del esqueleto del sitio utilizando etiquetas estructurales de HTML5 (header, nav, main, section, article, aside, footer) para asegurar la accesibilidad y el buen posicionamiento.
* Inserción de Contenidos: Integración de elementos multimedia, enlaces estructurados, tablas de datos y formularios de contacto accesibles.

### 2. Diseño y Estilos en Cascada (CSS)
Implementación de la capa de presentación visual sobre la estructura de marcas previamente definida:
* Práctica de CSS: Aplicación de estilos mediante hojas externas para garantizar la separación entre contenido y presentación.
* Modelo de Cajas: Control exhaustivo del espaciado mediante la manipulación de márgenes (margin), rellenos (padding) y bordes (border).
* Selectores y Tipografía: Uso de selectores avanzados (clases, identificadores, pseudo-clases) y personalización de fuentes, colores y fondos.
* Posicionamiento: Uso de técnicas de diseño para alinear y distribuir los elementos en la interfaz gráfica del navegador.

### 3. Sistemas de Gestión y Estructuración de Datos
Introducción a los lenguajes de marcas orientados al almacenamiento, intercambio y validación de información:
* Jerarquía de Datos: Creación de estructuras de árbol mediante lenguajes de marcas extensibles para la representación de datos independientes de la plataforma.
