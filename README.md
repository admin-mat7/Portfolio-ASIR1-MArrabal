## 📚 Bloques Técnicos por Módulos (Evidencias)

### 🖥️ 1. Implantación de Sistemas Operativos (ISO)
Administración avanzada de sistemas operativos libres (Linux/Ubuntu) y propietarios (Windows Server), gestión de usuarios, permisos, almacenamiento y automatización de tareas de sistema.

* **Evidencia destacada 1:** `Automatización con Scripts en Bash / PowerShell`
  * *Descripción:* Creación de scripts orientados a la automatización de copias de seguridad periódicas y monitorización del espacio en disco.
  * *Enlace al archivo/carpeta:* [[Ver Scripts](./ISO/scripts/)]
* **Evidencia destacada 2:** `Configuración de Servicios de Directorio`
  * *Descripción:* Despliegue de un entorno de dominio utilizando Active Directory (Windows Server) y gestión de Directivas de Grupo (GPOs).
  * *Enlace al informe/capturas:* [[Ver Documentación](./ISO/active-directory/)]

---

### 🌐 2. Planificación y Administración de Redes (PAR)
Diseño de arquitecturas de red locales, direccionamiento IP (IPv4/IPv6), subnetting con VLSM, configuración de conmutadores (Switches) y enrutadores (Routers), y gestión de VLANs.

* **Evidencia destacada 1:** `Diseño de Red Corporativa en Cisco Packet Tracer`
  * *Descripción:* Proyecto final de red que incluye segmentación de red mediante VLANs, enrutamiento inter-VLAN (Router-on-a-stick) y servidores DHCP/DNS.
  * *Enlace al archivo `.pkt`:* [[Ver Laboratorio PAR](./PAR/)]

---

### 🗄️ 3. Gestión de Bases de Datos (GBD)
Diseño lógico y relacional de bases de datos, implementación de esquemas con lenguaje SQL (DDL/DML) y optimización de consultas complejas.

* **Evidencia destacada 1:** `Diseño y Creación de una Base de Datos Relacional`
  * *Descripción:* Diagrama Entidad-Relación y script `.sql` de creación de tablas para la gestión de inventario informático, aplicando restricciones de integridad.
  * *Enlace al proyecto:* [[Ver Base de Datos](./GBD/)]

---

### 📄 4. Lenguajes de Marcas y Sistemas de Gestión de Información (LMSGI)
Estructuración de información digital mediante XML, validación de datos (DTD/XSD), transformación de documentos (XSLT) y desarrollo de sitios web semánticos con HTML5 y CSS3.

* **Evidencia destacada 1:** `Desarrollo de Sitio Web / Este propio Portfolio`
  * *Descripción:* Aplicación de los conceptos del proyecto de LMSGI para la maquetación y diseño fluido usando estándares web.
  * *Enlace:* [[Ver Código Fuente](./LMSGI/)]

---

### ⚙️ 5. Fundamentos de Hardware (FH)
Análisis de componentes físicos, arquitectura de computadores, ensamblado de equipos, sistemas de tolerancia a fallos (RAID) y entornos virtuales de pruebas.

* **Evidencia destacada 1:** `Laboratorio de Virtualización y Sistemas RAID`
  * *Descripción:* Configuración de arrays de almacenamiento redundante (RAID 1 y RAID 5) sobre entornos virtuales con Oracle VM VirtualBox.
  * *Enlace:* [[Ver Práctica Hardware](./FH/)]

---

### 💼 6. Formación y Orientación Laboral (FOL)
Estudio de los derechos laborales, prevención de riesgos laborales (PRL) específicos del sector tecnológico y desarrollo del perfil profesional.

* **Evidencia destacada 1:** `Plan de Prevención de Riesgos en un Centro de Datos`
  * *Descripción:* Análisis de riesgos físicos y ergonómicos en entornos de administración de servidores y desarrollo de medidas preventivas.
  * *Enlace:* [[Ver Documentación FOL](./FOL/)]

## 🧠 Reflexión Global y Evolución
El paso por este primer año de ASIR ha supuesto un cambio radical en mi comprensión de las infraestructuras informáticas. Al comenzar el curso, mis conocimientos de Linux y Redes eran básicos; hoy en día me siento capacitado/a para estructurar subredes complejas, administrar servidores en producción y automatizar flujos de trabajo repetitivos. 

El mayor reto técnico ha sido asimilar la lógica del enrutamiento y el subnetting VLSM en PAR, solucionado a base de laboratorios y perseverancia. De cara a 2º curso, mi objetivo es profundizar en la ciberseguridad, administración avanzada de servicios web y tecnologías de contenedores (Docker/Kubernetes).
